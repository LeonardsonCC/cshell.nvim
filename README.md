# EasyCmd
A plugin that just run the current file to the shell and shows the output in a popup

# Requirements
- Plenary

# Install
Using Packer
```lua
use {
    "LeonardsonCC/easy-cmd.nvim",
    requires = "nvim-lua/plenary.nvim"
}
```
